#CRM MEAN APP
<h3>Author:</h3>
Jocelyn Alsdorf

<h3>Description:</h3>
This is a MEAN application that has a token based login/logout system. CRUD functionality exists for users. MongoDB is the backend and the db is hosted via modulus.io.

<h2>Dependencies:</h2>

Nodejs
Angular 1.3.8
Angular-route 1.3.8
Angular-animate 1.3.8

"bcrypt-nodejs": "0.0.3",
"body-parser": "^1.14.1",
"express": "^4.13.3",
"jsonwebtoken": "^5.4.0",
"mongoose": "^4.1.11",
"morgan": "^1.6.1"
